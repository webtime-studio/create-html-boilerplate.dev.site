Добро пожаловать в документацию по Create HTML Boilerplate!

## Навигация по сайту
Оглавление разбито на разделы и темы. В дескопной версии меню находится слева и навигация по странице справа. На мобильном же устройстве навигация находится внизу экрана.

## Чего-то не хватает?
Если у вас есть идеи по поводу того, что какая-то часть функционала не покрыта, или что-то можно улучшить, напишите нам!
